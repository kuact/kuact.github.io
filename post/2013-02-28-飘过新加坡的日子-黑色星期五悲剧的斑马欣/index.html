飘过新加坡的日子 — 黑色星期五，悲剧的斑马，欣慰的亚马逊退货

2013-02-28
新加坡

Contents

      • 衣服，鞋子的大众尺寸
      • 亚马逊退货
    • Tips

本来这悲剧和新加坡打不上一杆子关系，只是自从两年前玩上了黑色星期五，这海淘风是愈演愈烈。临近一个月就开始眼巴巴的盯着黑色星期五各类促销网站，咱人不在米国，但也心痒痒的想在这个特别的打折日子里分一杯羹。知道黑色星期五也是群里整天叽叽喳喳的BT富豪团们，天天嚷着退群，却大把银子从米国捞便宜货。普及一下，所谓的黑色星期五就是指在米国每年感恩节Thanksgiving Day（11月的第四个星期四）之后的第一天开仓放价。那种Walk in的打折狂潮很多从星期五的0点就开始了。要知道感恩节（星期四）那一天是法定假日（通常星期五也没人想去上班）。于是和咱过年一样，大家高高兴兴吃完饭，打打牌；稍事休息就一同驾车去沃尔玛，梅西百货等各大知名店铺蹲点。等0点钟声一敲响，齐刷刷蜂拥而入。就剩一句话：好货不多，要抢从速。我等虽站在旁边流口水但也没有闲着，打鸡血地守在各大电子商务网站（啊，感谢万能的2000年电子商务泡沫），直到把眼睛挑花，黑白颠倒，才心满意足的睡过去。

这世界上不是只有中国人眼红米国的低价，蓬勃发展的海淘证明，很多国家都眼红。新加坡还更进一步，直接上了主力：vPost （官方新加坡邮政旗下中转物流）。接着有着银行支持的打折团也来了，Citybank花旗银行挑上了borderlinx，而Standard Chartered渣打银行直接去撑vPost。接着冒出了许多中转物流，比如这个打酱油的comGateway。除了comGateway用过，其他的都没有玩过，但就用comGateway这家就让我悲剧到吐血，并信誓旦旦的要把comGateway这坨屎悬挂半年。它家赚钱的独门秘笈就是不管如何算实重还是体积重，最后都会多算重量。我在亚马逊买一条快干裤，买回来裤子已经大到吐血了，结果comGateway居然把这个0.5kg的小盒子算成了2.5k，活生生的让它的运费超过我的裤子. 只能说，新加坡海淘唯一好处就是超过400 SGD才征税，统一征收7%。

  1. vPost 走EMS万国邮政路线，运输质量还是有一定保障的，但是总体评价就是慢
  2. Borderlink 价格比vPost贵，物流合作伙伴是DHL，比vPost快，但骂它家的人也很多
  3. comGateway 一坨屎，继续悬挂

唉，总而言之，还是华人物流与海淘强大，国内扎堆的中转物流，我就用过两家：一家是天翼，它刚冒头没有多久我就用上了它，居然现在还活着，看来确实是过的风生水起。之所以不再用天翼，一个是因为它要先充值，后填单，搞得每次都有钱剩在它家帐号里（好天真的营销策略）。连comGateway都是货到算钱再付费的，你家非让我先充值，没有这样当水鱼的。二来是它和万国邮政关系不够硬，非要上传身份证。在天朝这个充分互相不信任的社会里，身份证是一个很敏感的东西好不好。就这样直接把天翼抛弃了，后来又被群里忽悠，用起了海淘中的战斗机：斑马物联网。我用天翼，因为图免税州的便宜，结果前年的黑色星期五，它家的免税州爆仓，一件货拖了将近一个月才发到我手上。但总的来说还是靠谱的，没有给我丢过货；反倒是战斗机一上来就把我的货给丢了，又是一段血泪史啊。

话说去年黑色星期五的一单，寄到斑马的仓库选择合箱发货。结果连续提单，连续退单，理由只有一个：红外线扫描发现里面有香水！土鳖也知道有没有买香水好不。好吧，硬说我的箱里有香水，我就选择拆箱发货（问题的关键）。拆好后提单几天又没有动静，于是苦口婆心的邮件催电话打。这个时候斑马客服充分发挥了脸皮厚比城墙的优势，拖的你是筋疲力尽，完全没有商量。于是持久战又打了两个星期，最后的回复结果：货丢了，而且就丢在自家的仓库里！于是又开始漫长的马拉松索赔。承诺一个月处理完，结果又忽悠不断。先是说突然找到了拖你几天，结果又突然没找到。2013年2月26日，历时3个月，我这单终于赔了，可是一查帐号：赔的提单时的申报金额！这回我是彻底哑巴吃黄连了，啥血都要自己吞了。

长江后浪推前浪，前浪死在沙滩上，总不能让前浪死的不明不白：

  1. 客观的说，斑马的物流体系还算是靠谱的。关键操作是减少各种潜在风险的环节。最低风险的转运方式就是原单转运，要省点运费的话就合单转运，我这种拆单的就是最大风险。
  2. 合箱：一箱的鞋子不要超过2双，一箱一个包，一箱一件电子产品（kindle也算），一箱总重不要超过10lb
  3. 申报价值：只要不是包，鞋子；绝大部分都不会被税，申报的价值你看着办，但如果丢了就是我的下场
  4. 衣服，鞋子大小一定要看网上的经验。网站上的尺寸示例都偏大（亚马逊也不靠谱）。

亚马逊退货

去年黑色星期五，无限不顺。买的一件大衣，照着亚马逊的示例尺寸量的，结果好歹寄回来了，发现大了不是一码两码。实在太大，想送人都没有人选。被逼到最后一步退货，花了几百块让衣服围着地球转了小半圈。但不得不说亚马逊的退货服务真是相当的赞呀，完全自动订单处理，无需人工介入。
1. 登录亚马逊帐号
2. 找到购买的订单，选择要退货的商品，随便找个退货理由，提单（如果超过退货期，是不会让你提单的）
3. 亚马逊会给你发一封邮件（注册邮箱），把退货标签打印出来（mailing label），第一页和货物放在一起寄回来，第二页是美国的退货地址
4. 到淘宝找一个EMS中介 （顺丰不发那个退货地址），EMS要填电话，填亚马逊的客服电话号码：1-866-216-1072

于是就开始耐心等待吧（可能会出现几次投递不成功现象），只要亚马逊收到货，大约2/3天后，订单全款会退到你的信用卡里（EMS这段邮费它是不会退的）。不过亚马逊的黑色星期五也玩忽悠，这件大衣黑色星期五当天只有30%折扣，结果我退完货回来发现居然打五折。

其实海淘不一定适合每个人，除了一大堆注册，购买手续，信用卡与流程，还要担惊受怕的享受中转物流中的种种神奇，时间拖延的煎熬；甚至最后到货后还要感谢各路神仙千万不要买错了。并且一上中转物流的贼船，就如水鱼任人宰割，货在十万八千里之外，你只有孟姜女哭的份。哭？哭也算仓储费的哦。虽然理论上说来，谈判的天平是平等的，中转物流也都在小心翼翼的经营信誉。但是在这个信誉及其脆弱的天朝，信息的不对等让天平一开始就弯了。要真碰到打一枪换一炮的，你连哭的份都没有。所以如果你购买的物品没有大大超出你的预期，千万不要随便上船啊，这船票贵呀。
