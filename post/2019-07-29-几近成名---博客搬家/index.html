几近成名 – Wordpress博客搬家Hugo+Github Page

2018-07-29
几近成名

Contents

    • 方案来了
    • Tips

很久很久以前，我的blog在微软MSN Space上写，后来微软把MSN Space关了，我又稀里糊涂的用上了wordpress。还信心满满的买了域名，不知道从哪里找来的免费的空间部署了一把wordpress，倒也其乐无穷。

然后，然后免费空间就挂了；雪上加霜的写blog的积极性因为这样那样的原因备受打击。于是乎，心理有一种在上面种草，任其放任自流的态度。

可怜我的域名一直还在，于是我就将域名外挂回了wordpress空间。谁知道挂一个DNS域名也是要收费的…. 心理又被小小打击了一把。

更加无耻的是，wordpress还充分利用我blog的内容来打广告。本来我写blog的初衷就不是为了挣流量，现在有一种满满被利用的感觉。

于是wordpress挂域名续费到期，我铁了心不再给它买单，是的，wordpress，我要抛弃你了。今天看到一句话：CHANGE IS ONLY CONSTANT IN LIFE，解释为中文就是生命不止，折腾不息….

一直苦于没有很好的规划和时间上的安排，blog就这样悄无声息的断了几个月。好在，访问的人本来就很少，大部分人基本获取到了信息就迅速撤离，我觉得自己缺少对别人认真的态度。

无论如何，这些年写的软文积攒了不少，就这样放弃掉有些可惜，于是又打起精神开始找替代方案。唯一的前提就是：我写只为我自己…..

于是乎贴图已经不再是必需品，它要越简单越好，越容易维护越好。并且一定要免费，免费，免费…

方案来了

  • 博客平台 hugo 号称世界上最快的静态页面生成平台，嗯嗯，的确如此。外加用go语言写的，在windows平台上很容易就可以上手。

打怪指南

  • 博客语言 Markdown Markdown 语言格式很方便，也很容易掌握

打怪指南

  • MD编辑工具 Visual Studio Code(VSC), 微软家荣誉出品，而且还是免费的。 写代码好用，写MD也是。开preview小窗，可以实现所见即所得。

打怪指南

  • 网站部署Github Page Github本来做代码托管的，为了方便大家，Github出了Page功能专门用来挂静态网页了；所以免费空间就此而来。将来Github Page被墙，你有着MD文件，还怕找不到地方部署么？

打怪指南

  • Blog搬家 这里走了很多弯路，因为总想着找个插件一劳永逸来个一键搬家，结果碰了一鼻子灰。饭还是要一口一口吃的，罗马岂是一天建成。
  1. Wordpress导出XML备份
  2. 用XAMPP搭建本地blog 打怪指南
  3. 安装wordpress转换MD插件 打怪指南 我是最后用命令行成功装MD的
  4. 使用自动发布脚本 打怪指南
  5. 挂域名 打怪指南
  6. HTTPS跳转 打怪指南

Tips

  • 注册一个github账号
  • 注册一个域名（如果你还没有的话）
  • 安装github, go编译器
  • wordpress转MD插件依然不是很完美，每一篇文章都会单独生成一个MD文件，还是需要人工介入微调一下的
