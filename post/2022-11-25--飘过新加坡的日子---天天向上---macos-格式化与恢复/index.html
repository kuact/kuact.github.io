– 【1】题记 –

2010的macbook air连Ella都嫌弃，只好准备处理掉。

我对macOS向来不感冒，临时抱佛脚去清理硬盘。

–【2】硬盘格式化 –

Disk Utility自带DoD 5220.22 M算法来擦除硬盘；但只针对外置硬盘；然后老款macbookair也没有安全芯片。

只好曲线救国找第三方格式化软件

  • BitRaser
  • AweEraser

制作启动盘，刷NIST 800-88 Purge；呵呵，洗白了。

– 【3】系统重装 –

苹果除了工程设计好之外，系统重装和恢复真是一个大大的赞。

开机启动没有传统的的Bios界面，几个快捷键：

  • OPTION: 选择启动硬盘
  • COMMAND + R：进入macOS Utilities
  • OPTION + COMMAND + R：进入互联网系统恢复

这个互联网恢复简直是神操作，它会提示你提供WIFI登录。

macOS Utilities

里面有一个最重要的Disk Utility对内置硬盘进行格式化

  • View -> Show All devices
  • 找到内置硬盘，选择erase
  • 选择格式APFS，其他默认

接着就选install macOS一路OK。

制作macOS启动盘

前提是你首先可以进入macOS

  • 准备8GB U盘，用disk utility格式化
  • 登录官网，下载对应OS到applications
  • 按照官网指南操作即可

命令行： sudo /Applications/Install\ macOS\ High\ Sierra.app/Contents/Resources/createinstallmedia –volume /Volumes/MyVolume
