– 【题记】–

好吧，快5年没有回国；这次回去顺带把多年前压箱底的wii捞了回来。嗯嗯，我就是个垃圾佬，尤其对这些电子类的东西死活不放。就如母亲对一切的浪费都深恶痛绝，表现在食物上可是硬往我碗里噻呀。有时候我也很纳闷这种捡了芝麻，丢了西瓜的行为。你老人家要是因为省这半毛钱，摔一跤；或者吃过期的剩菜拉两天肚子；这不是妥妥金钱与身心的消耗么。更让人发指的是，家里修个马桶也要找我上，打心底这个学计算机的应该一口价全包。呵呵，好吧，我也就是抱怨一下；潜意识里其实在母亲这边，只是我自己不知道而已。

转回来，WII这货当年也是横扫全球，那些经典的体感游戏虽然只有480P的分辨率，但完全不负家庭欢乐的经典。反正都捞回来了，那么就开始折腾吧…..

基本需求：

  • 抛弃光盘
  • 只用SD card
  • HDMI 输出
  • 四个遥控（海鲜市场）

– 【流程】–

主线流程：

  1. 开WIFI，升级最新的WII版本4.3（WIFI只支持2.4GHz）
  2. 设置DNS安装homebrew
  3. 格式化SD卡，下载如下插件，并拷贝到SD卡app目录下
  • d2x-cios-installer
  • LoadPriiloader
  • priiloader
  • some-yawmm-mod
  • usbloader_gx
  1. SD卡根目录创建wad目录，并下载：USB-Loader-GX-UNEO_Forwarder_5_1_AHBPROT-Wii.wad
  2. SD卡根目录创建wbfs目录，用于wbfs格式wii游戏
  3. 用WiiBackupManager转iso成wbfs格式

注： WII现在的主打版本号有4.3J （日版），4.3U（美版）和4.3E (欧版)。

– 【homebrew】–

homebrew 类似第三方平台，插件只能安装在这里

  1. d2x-cios-installer

    • ios是WII操作系统称呼，cios就是定制系统；装这个插件就是把定制os写到WII里去；安装成功后可以从SD卡删除
    • 安装如下几个定制OS
      • Select cIOS
      • Select cIOS base <38>
      • Select cIOS slot <248>
      • Select cIOS revision <65535>
      • Select cIOS
      • Select cIOS base <56>
      • Select cIOS slot <249>
      • Select cIOS revision <65535>
      • Select cIOS
      • Select cIOS base <57>
      • Select cIOS slot <250>
      • Select cIOS revision <65535>
      • Select cIOS
      • Select cIOS base <58>
      • Select cIOS slot <251>
      • Select cIOS revision <65535>

  2. LoadPriiloader

    • 在homebrew中启动riiloader

  3. priiloader

    • WII系统备份，防止变砖
    • 游戏解区

  4. some-yawmm-mod

    • 可以安装wad插件

  5. usbloader_gx

    • 加载SD卡中的WII 游戏

– 【usbloader_gx】–

讲重点

  • 如果用USB，使用WII底下的接口，否者不认
  • 如果用SD，使用SDHC卡，不要用SDXC；即使是SDHC卡，也会跳卡。form factor你可以直接用microSD+SD壳即可
  • 用FAT32格式，其他格式usbloader不支持。
  • 也可以格式化一个USB硬盘成wbfs格式专门装游戏，但是一旦wbfs格式，windows是不认的， 只能用WiiBackupManager查看

  • 设置上

    • Loader setting -> Return to -> USBloader GX
    • Language setting -> English (只要下载的美区的ISO)
