Quest 3其实刚出不久就入手了，想着来个家庭体验篇，却一直拖延到现在。最近很忙，嗯，应该这大半年都超级忙；但是心里却很难过。就是那种压力极大，成就感归零的感觉。嗯，是老板的leadership出现了问题；并且在压力面前越来越离谱。老板在出现错误的时候没有及时纠正，弄整个团队都跟着遭殃。所谓一地鸡毛，大约就是现在的样子。想着一走了之，却被各种困扰；在客户面前还得打肿脸充胖子，硬挺着。然后最近我最得力的同事移民澳洲；一下心就没底了，真不知道我还能坚持多久。

  • 可以用adb 无线安装应用

  • 可以用referral code

– 【SideQuest】–

Quest 3 基于Android； 是andriod就一定有一个神器adb。而sidequest简而言之就是图形化的adb（apk应用）；所以就直接掠过sidequest，只要能开启adb就行。至于国内的VR助手，87VR什么的，效果和sidequet同款。

– 【开发者模式】–

要用adb，就要开启开发者模式；流程如下：

  • 手机下载Meta Quest应用
  • 用邮箱注册（不用facebook账号），开启2FA
  • 网页登录Oculus开发者
  • 创建组织
  • 验证账号
  • 回到手机Meta Quest / 设置 / 开启开发者模式

开发者模式对quest 2， quest pro用同样适用；所谓一通百通，adb一开就天下无敌了。

– 【ADB模式】–

  • 下载Oculus ADB驱动（windows版本）
  • 实测有线/无线都可以adb安装了APK了（呵呵，资源自己找）

注： 如果出现obb文件上传权限错误，可以先用adb安装apk本体，然后quest 3启动游戏。游戏启动后，其会自动创建obb空目录。这个时候在adb push obb文件到系统自动创建的空目录即可。

比如：

adb push “xxx.obb” /sdcard/Android/obb/xxx

– 【无线串流】–

如果追随官网搞有线串流，各种设置简直让人抓狂。所以这里直接跳过上无线串流。那么问题来了，无线串流哪家强？嗯，以前是virtual desktop；但现在quest商店里横空出世steam应用；跳过oculus中间层，直接启动steam不香么。 你要还没买virtual desktop，现在可以直接steam走起。

  • Half-Life: Alyx 了解一下

– 【推荐码】–

  • 新加坡没有推荐码
  • 所以要想拿30美金的推荐码，先VPN到美区地址
  • 再用推荐码链接注册Quest账号（美国地址）
  • 最后激活quest 3

如果已经激活过quest 3，要在oculus网站上删除quest 3设备。这样会对quest 3设备重置；然后在你的quest 3上进行factory reset。

注：

  • 设备重置不影响此账号在商城上购买的游戏。
  • McAfee有VPN服务，考虑一下。

– 【游戏购买】–

  • 日区购买游戏最便宜（目前汇率）
  • 外加25%游戏折扣码（google 之）
  • VPN到日区地址，登录quest 商城购买（可paypal）
  • 退出VPN，重新登录，游戏就在你库里面了。
