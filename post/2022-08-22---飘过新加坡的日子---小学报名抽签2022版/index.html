题记

热门区域Clementi沙盘

报名是分阶段的，从1到2CS，学生名额累次递减。

这个区域有4所小学

  • Nan Hua
  • Qifa
  • Clementi
  • Peitong

2B

2B的报名窗口期只有2天，报名期间不会公布实时报名情况。所以只能以滚雪球历史数据查询作为依据。Clementi有一所Top 10 南华(Nan Hua)，另外一所算第二梯队的启发（Qifa)。

2B 南华妥妥的1km SC抽签，所以大部分1km的SC都会去冲南华；这样拿2B的PR们就会去报启发（Qifa）。然后基本上没人会把2B浪费在Clementi和peitong上。

名额/报名数

  • Nan Hua： 20 / 38
  • Qifa： 44 / 22
  • Clementi： 60 / 0
  • Peitong：58 / 0

2C

2C的报名窗口期有4天，你可以报很多次，但MOE只以你最后一次为基准。刺激的是MOE会每天公布前一天的报名数据（最后一天没有公布）。所以大家都是憋足了劲留到最后一天男儿当入樽。

名额/报名数

  • Nan Hua： 40 / 191
  • Qifa： 111 / 131
  • Clementi： 183 / 71
  • Peitong：175 / 34

南华1km内5选1，启发也要1-2km内抽签。分化就从这里开始了，如果家里位置好，南华和启发都是1km以内，如果2B没拿到；求稳就会争启发。那些启发1km以外的，南华1km以内的反倒就尴尬了。因为2B已经失落了，这个时候还要硬着头皮继续搏南华。

为什么不在2B求稳去启发呢？答案也很简单，房子如果租的，你肯定会挑南华和启发1km。这样2B无论如何都会搏南华，然后2C冲启发。但是如果你买在南华1km而距离启发差了一丢丢，你会放弃家门口的南华去启发？要知道启发隔了一条AYE，上学放学怎么麻烦怎么来。

另外今年南华旁边CLEMENTI NORTHARC落成，平白增加了1000个单位加入竞争，他们基本上一条路走到黑。2B没中，2C继续；心态要稳住，后面还有Clementi保底。

2CS

其实真正保底的是Pei Tong；Clementi居然2km SC要抽签。PR在2CS基本上没戏；所以PR只能在2B的启发和2C的Clementi里面选。

至于SC，心态真的很重要；如果2B和2C都没进，家长会手抖选Pei Tong；虽然Pei Tong和Clementi其实差不多；但Clementi的位置要好一丢丢。

名额/报名数

  • Nan Hua： 0 / 0
  • Qifa： 0 / 0
  • Clementi： 112 / 125
  • Peitong：143 / 94

可怜天下父母心，你们真的尽力了。
